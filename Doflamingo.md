# 𓂀 Doflamingo WhatsApp Bot v1.0.0 𓂀

## ⚡ Installation

1. Clone the repository:

```bash
git clone https://github.com/LORD_OBITO_DEV/DOFLAMINGO-XMD-V1.git
cd doflamingo-xmd-v1
```

2. Install dependencies:

```bash
npm install
```

3. Run the bot:

```bash
npm start
```
